Our Plan

Abstract: Compare the performance of X-Pilot controllers trained with Artificial Neural Network and with Fuzzy Logic when both use Genetic Algorithm to learn their parameters.

We are going to evolve two controllers in X-Pilot, one with Neural Network and the other with Fuzzy Logic. For both approaches, we will be using Genetic Algorithm in the learning process. For the controller with the Neural Network, we will use Genetic Algorithm to learn the weights associated with the network. On the other hand, for the Fuzzy controller, Genetic Algorithm will help us learn the thresholds for the Fuzzy sets.

We plan on having the two controllers compete individually against a well-trained aggressive bot made by Cam and Nikesh to evaluate the performance. In addition, we will also have a competition between the neural network and the fuzzy controllers for head-to-head results.

Our Map


Neural Network + Genetic Algorithm

We use a neural network with 18 inputs and 3 outputs to control the X-Pilot agent. The inputs includes distances to walls in all directions, velocity of the agent, distance and angle of the most dangerous bullet, ..., while the outputs include thrust (True or False), shoot (True or False), and turn (an angle from -20 to 20 degree). All inputs are normalized to between -1 and 1.


At the moment, there are no hidden layers, thus the number of weights in the neural network is 54. These weights are learned using Genetic Algorithm, with a population of 100 chromosomes, each with 324 genes. Each weight in the neural network is represented by 6 genes, thus 54 weights fit in a chromosome of 324 genes.


We use a standard Genetic Algorithm where each chromosome in the population is tested and assigned a fitness, which is calculated by how long the agent survives plus a bonus if the agent kills the opponent. The population then are selected using roulette wheel method and evolved using two point crossover and mutation with a chance of 0.1. Each chromosome is trained against an aggressive and accurate opponent provided by Nikesh and Cam (thank you!).




Fuzzy Logic + Genetic Algorithm

Our Fuzzy/GA bot uses a genetic algorithm to evolve both the bots production system and the membership functions for the Fuzzy Logic. We use 311 bit chromosomes, translate portions of the chromosome from binary to base ten, and allow each portion to represent a certain part of our code. We have four membership functions, which use 146 bits of the chromosome. Each membership function has a portion of the chromosome to calculate: where the degree of membership (DOM) is 1.0 for low, what the slope of the low DOM portion is as it decreases to 0, where the medium DOM starts, the slope for DOM as it increases and decreases, where the high DOM starts, and what the slope of the high DOM is as it increases to 1.

Experiments

Neural Network + Genetic Algorithm

We started off by training the agent with 18 parameters with the fitness increasing for every frame it survives by 1. After many generations, the chromosomes were converging on the same strategy of hide and seek. The enemy we trained against was an aggressive agent that would hunt enemies by flying around the map until enemies appeared on screen and accurately aim and kill its enemy. Since our agent’s (Nettie’s) fitness was based on the number of frames it survived, it would hide in the corner of the map and spin in circles until the enemy appeared nearby. The most fit chromosomes learned to not move at all, so that it would not be found and killed.


To counter this we added in a -1 fitness to all frames with a speed equivalent to 0. This ensured that the agent would only have a fitness above 0 if it moved throughout its lifetime. The unintended consequences of this led to Nettie learning to shoot instead of thrusting. Every time an agent shoots, it accelerates backwards slowly. Instead of learning to aim away from walls and thrust to avoid walls, Nettie learned to aim at the closest wall and shoot at it to avoid moving closer to the wall without thrusting. We tested this model with 256 chromosomes in one population and 512 chromosomes in another population.


Once again, we decided to attempt to change the behavior to learn to thrust. Our last test altered the fitness by dividing the chromosome’s fitness in half if it did not thrust throughout its lifetime in addition to the previous fitness changes. We created a global variable set to ‘True’ or ‘False’ to check if the agent thrusted during its lifetime. This resulted in Nettie continuing the previous strategy of shooting to move each frame until the enemy was on the screen. Once the enemy was on the screen, it would thrust very powerfully into the wall directly ahead.



Fuzzy Logic + Genetic Algorithm

Originally, we were only going to evolve our degree of membership functions and choose production system values ourselves, but decided that it would most likely have led to NN bot performing far better than Fuzzy. Having a production system created without the help of a genetic algorithm would have caused the Fuzzy bot to have an external factor affecting its performance (our decision making on deciding numbers for a production system) and would have made for an unfair comparison between NN bot and Fuzzy bot.


We originally had 146 genes in each chromosome, though after updating the code to have an evolving production system we ended up with 311. We took our fuzzy system from a previous code and altered the wall danger for higher performance. We had also originally wanted to reward a smaller amount for kills, but doubled this amount to encourage a more aggressive bot. For the first few generations, our bot was frantic. It performed decently with aiming and shooting, but would thrust out of control and into walls. After a few generations, it became noticeably more in control. The bot would thrust towards a wall, turn until its back end was facing the wall, and thrust away.


After 287 generations, every chromosome was identical. The bot had good aiming, shooting, and wall avoidance. We altered the code to use only the best chromosome and played it against NN bot.
